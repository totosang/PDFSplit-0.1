# PDFSplit-0.1
Advanced browser-based PDF tool: Split, merge, rotate, and rearrange pages via smooth drag-and-drop. Features high-quality previews and instant export to Merged PDF, ZIP, or JPG images.เครื่องมือจัดการ PDF ครบวงจรบนเว็บ: แยก-รวมไฟล์, หมุนภาพ และจัดเรียงหน้าใหม่ด้วยระบบลากวางที่ลื่นไหล ส่งออกเป็น PDF ไฟล์เดียว, แยกไฟล์ ZIP หรือรูปภาพ JPG ได้ทันที
